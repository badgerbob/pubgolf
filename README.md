# Pub Golf PWA

This is a lightweight installable web app version of the pub golf score card.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `sw.js`

## Run locally
You can host it with any static hosting service.

Examples:
- Netlify
- Vercel
- GitHub Pages
- any shared web hosting

## Quick test on your computer
If you have Python installed:

```bash
python3 -m http.server 8080
```

Then open:
`http://localhost:8080`

## Install on iPhone
1. Deploy the folder online.
2. Open the site in Safari on iPhone.
3. Tap Share.
4. Tap Add to Home Screen.

## Notes
- Scores save in browser local storage.
- Works well as a simple PWA.
- Offline support is included after first load.
- Pub images are bundled locally from the venues' public website/listing photos.
